About NotesApp

Simple & Powerful Notes Management System

NotesApp is a modern web application built using Flask and SQLite. It helps users create, manage, update, search, and organize their notes efficiently.

Features

  • User Registration & Login
  • Create, Update & Delete Notes
  • Search Notes Functionality
  • Secure Password Hashing
  • Responsive Bootstrap UI

Technologies Used

Python Flask SQLite HTML CSS Bootstrap 5
